실무에서 잘 쓰는 git 용어
   
```
git clone   
git init   
git remote -v   
git remote add 원격레포 원격주소   
git remote remove 원격레포   
git log --oneline --graph --all   
git branch   
git push 원격레포 브랜치   
git checkout -b 브랜치이름    
git stash   
git stash list   
git stash save "이름"    
git stash apply   
git stash apply stash@{2}    
git stash drop   
git merge 대상브랜치   
git revert --soft 해쉬      
git revert --hard 해쉬    
git log   
git fetch 원격레포 브랜치   
git pull 원격레포 브랜치   
```
